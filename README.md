# Basic git commands:
- `git clone someUrl` -copies a git repo from github into your local machine
- `git pull origin master` -pulls down the latest code from the master branch
- `git checkout -b myNewBranche` -creates a new branch called myNewBranch locally
- `git checkout myNewBranche` - checks out an existing branch named myNewBranche
- `git add .` -stages all saved changes
- `git commit -m "This is my message"` -commits all staged changes and attaches a message
- `git push origin myNewBranche` -pushes all commits to github on that branch
